What's it all about?

phpFUS acts as a real-time virus and malware scanner for PHP web applications, utilizing both our own virus definitions table and optionally the clamav daemon.

What happens?

phpFUS looks for all $_FILES requests and scans each file individually before your application runs. If the file is malicious it is quarantined and made unavailable to your application, protecting you and your server

Coming soon...

We will be introducing a global IP blocking system, this will automatically block all attacker IP addresses known by phpFUS.

It'll be an opt-in service. The way it works is simple, phpFUS blocks an attacker, it submits the IP address and attack type of the attacker to PHPScanner HQ, all phpFUS installs then sync with the phpFUS HQ.

What is the phpFUS HQ? - well, at the moment it doesn't exist, your instance of phpFUS runs completely stand alone and has no communication with us, but in the future our aim is to develop a network allowing phpFUS's everywhere to share knowledge of known attacker IP addresses.
